# 🧮 Calculadora Simples em JavaFX

## 📌 Descrição do Projeto

Este projeto foi desenvolvido como uma atividade acadêmica com o objetivo de implementar uma **calculadora simples com interface gráfica**, além de realizar **testes de sistema** para validar seu funcionamento.

A aplicação permite realizar operações matemáticas básicas de forma intuitiva por meio de uma interface gráfica semelhante a uma calculadora convencional.

---

## 🎯 Objetivos da Atividade

* Desenvolver uma calculadora com interface gráfica.
* Implementar operações matemáticas básicas.
* Aplicar conceitos de organização de código.
* Realizar **testes de sistema** para validar o funcionamento da aplicação.

---

## ⚙️ Tecnologias Utilizadas

* **Java**
* **JavaFX** → utilizado para criar a interface gráfica da aplicação.
* **CSS** → utilizado para estilização da interface da calculadora.
* **JUnit** → utilizado para implementação dos testes.

---

## 🧮 Funcionalidades da Calculadora

A calculadora possui as seguintes funcionalidades:

* Adição (+)
* Subtração (-)
* Multiplicação (×)
* Divisão (÷)
* Inserção de números
* Inserção de números decimais
* Limpar visor
* Apagar último número digitado

---

## 🧪 Testes

Foram implementados **testes utilizando JUnit** para validar o funcionamento das operações da calculadora.

Os testes verificam se as operações matemáticas estão retornando os resultados esperados.
